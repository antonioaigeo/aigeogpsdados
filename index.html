Tudo o que Você Precisa

Funcionalidades pensadas para o trabalho de campo.

Dados em Tempo Real

Acompanhe sua latitude, longitude, precisão e o número de satélites em tempo real.

Descrição de Pontos

Adicione uma descrição detalhada a cada ponto coletado para uma organização impecável.

Exporte para CSV & KML

Com um clique, exporte seus dados para formatos compatíveis com qualquer software GIS.

Ideal para Trabalhos Acadêmicos

Perfeito para TCCs, dissertações e teses que envolvem trabalho de campo.

Rigor Metodológico

Exporte dados em CSV ou KML para incluir facilmente na sua metodologia, com registo de precisão para cada ponto.

Dados Organizados

A função de descrição de pontos garante que as suas anotações de campo não se percam, facilitando a análise posterior.

Custo Zero

Uma ferramenta gratuita e de qualidade que não sobrecarrega o orçamento do estudante ou do pesquisador.

Como Instalar

1

Faça o Download

Clique no botão de download para baixar o ficheiro aigeo_GPS_v1.2.1.apk.

2

Autorize a Instalação

No seu smartphone, encontre o ficheiro e toque nele. O Android pedirá permissão para "instalar de fontes desconhecidas". Autorize.

3

Pronto para Usar!

O ícone do AIGEO GPS Dados aparecerá na sua lista de aplicativos. Abra e comece a coletar!
